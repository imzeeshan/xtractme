# TailAdmin Assets Directory

## Setup Instructions

1. **Download TailAdmin HTML version** from https://tailadmin.com/
2. **Extract the following files** to this directory:
   - `css/style.css` - Main TailAdmin stylesheet
   - `js/` - JavaScript files
   - `images/` - Images and icons
   - `fonts/` - Font files (if any)

## Directory Structure

```
static/tailadmin/
├── css/
│   └── style.css
├── js/
│   └── (TailAdmin JS files)
├── images/
│   └── logo.svg (or logo.png)
└── fonts/
    └── (font files)
```

## Quick Setup (Using CDN)

If you don't want to download TailAdmin files, the templates are configured to use Tailwind CSS CDN as a fallback. However, for full TailAdmin features, download the HTML version and extract assets here.

## Alternative: Use Tailwind CSS CDN Only

The admin templates are configured to work with Tailwind CSS CDN if TailAdmin assets are not available. This provides basic Tailwind styling without the full TailAdmin design.
